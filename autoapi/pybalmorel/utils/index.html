Args:

db (GamsDatabase): The loaded gdx file symbol (string): The desired symbol in the gdx file cols (list): Your defined columns, will otherwise first try to find pybalmorel default column formats for the symbol or the raw columns from the gdx result_type (str): Is it a normal MainResults or a Optiflow Mainresults?
