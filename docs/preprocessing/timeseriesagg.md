# Time series Aggregation

The time series aggregation function uses dependencies that are not automatically installed with pybalmorel. riprepy (a python wrapper for [ripgrep](https://github.com/BurntSushi/ripgrep)) and the time series aggregation package [tsam](https://tsam.readthedocs.io/en/latest/). Install into your virtual environment with pip install:

```zsh
pip install tsam
pip install riprepy
```
